Shared React UI components for Von's dashboard and applications. Provides reusable components with consistent styling across Von's web interfaces.

## Installation

```bash
bun add @usevon/ui
```

## Usage

```typescript
import { Button } from '@usevon/ui';

const App = () => {
  return <Button variant="primary">Click me</Button>;
};
```
